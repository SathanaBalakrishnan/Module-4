4. **Sort by Values**:

5. **Display** the original and sorted dictionaries.
6. **End the program.**

---

## ðŸ§ªProgram
```
Dict={3:323, 1: 2, 6: 18, 4: 24, 2: 56, 5: 12}

print("Keys and Values sorted in alphabetical order by the key")
for k,v in sorted_items:
    print((k,v),end=" ")
```
